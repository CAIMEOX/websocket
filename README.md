# Websocket
Minecraft websocket
# How to build ?
You need download NodeJS,and install ws and fs.Then open cmd input "node $file".
# FastBuilder
create sphere:
``sphere <x;y;z> <Radius:int>``

create round:
``round <x;y;z> <Radius:int>``

create circle:
``circle <x;y;z> <Radius:int>``

create ring:
``ring <x;y;z> <Radius:int>``

and more....
